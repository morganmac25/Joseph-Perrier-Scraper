# Joseph Perrier Scraper

A Python project to scrape product data (name, description, type, images, videos) from the Joseph Perrier champagne website.

---

## Features
- Fetch and parse listing pages
- Scrape individual product pages
- Save products and media to a database
- Handles images and video embeds
- Deduplicates products automatically

---

## Requirements
- Python 3.10+
- pip
- Virtual environment recommended
